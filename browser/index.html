Engi

Engi is an open source HTML5 dataflow programming editor supporting web technologies like WebGL

Try it now!

Quick start

You can run it right here, but we recommend running Engi locally to use all the features such as recording. You'll need to install git and Node.js first. On the Mac, you can use Homebrew to do that. 

	$ git clone https://github.com/engijs/engi.git
	$ cd engi
	$ npm install
	$ node server

Then point your browser to http://localhost:8000/. Engi works best in the latest versions of Chrome and Firefox.

Introduction

Engi has over 200 plugins that you can combine visually to create logic, 3D graphics, cameras, materials, lights etc. You can play video and audio, write custom shaders, and a do a whole lot more. It's written in JavaScript with high performance in mind. For more information, see below or read more in the introduction and the README. It works in the browser so if you just want to get started, you can use it right here!

Engi is open source and licensed under the MIT license. The source code is at https://github.com/engijs/engi.

Editing graphs


Right click on canvas to insert plugins.

Drag connections from inputs and outputs

Press Play to run your graph.

Hold shift to delete plugins and connections.

Double click to add Presets.

Click and drag on the canvas to select plugins.

Collapse plugins by clicking on their icon.

Hotkeys


CTRL-C, CTRL-V, CTRL-X Copy, Paste, Cut

CTRL-A Select all

CTRL-B Hide/show Graph tree and info view

CTRL-L Hide/show console.

Space Play/Stop

Tab Search Presets

Backspace/Delete Delete selected plugins
